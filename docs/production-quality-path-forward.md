# Production Quality Path Forward

Date: 2026-05-09

## Purpose

This document defines the future quality bar for QuarterlineV2. It is a scaffold
until implementation begins.

## Quality Dimensions

QuarterlineV2 should eventually be judged across:

- Desktop installation and startup reliability.
- Local-first data safety.
- Confidential file handling.
- Analyst workflow ergonomics.
- Dense table and chart readability.
- Scenario accuracy and explainability.
- AI output traceability.
- Report assembly quality.
- Export fidelity.
- Offline behavior.
- Performance on realistic workspaces.

## Design Quality Bar

The product should preserve the design goals in
`docs/design/quarterline-v2-design-goals.md`:

- Industrial minimalism.
- Dark institutional workspace.
- Compact, high-density controls.
- 8px panel/control radius.
- Tabular numeric precision.
- Clear AI, map, stacking-plan, financial table, and scenario tiers.

## Future Rubric Template

Create a scored rubric later with categories for:

- Desktop app readiness.

- Report output readiness.
- AI assistance quality.
