# Analyst Test Script

Date: 2026-05-09

## Purpose

This file will hold future usability-test scripts for QuarterlineV2. It is a
template until a runnable desktop prototype exists.

## Test Goals

Future tests should answer:

- Can an analyst understand the desktop workspace quickly?
- Can they locate portfolio, market, property, scenario, and report tools?
- Do dense tables and visual modules feel trustworthy?
- Does report pinning match their workflow?
- Do local save/export states feel clear?
- Are confidential-file boundaries understandable?

## Session Template

1. Participant profile.
2. Setup and consent.
3. Scenario prompt.
4. Task list.
5. Observation notes.
6. Trust/friction moments.
7. Report-output reaction.
8. Post-test questions.
9. Follow-up decisions.

## Current Status

No V2 analyst test has been run.
