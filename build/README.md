# Build resources

`icon.ico` is the multi-resolution Windows icon (16, 24, 32, 48, 64,
128, 256), generated by `scripts/generate-icon.py` from V1's brand
identity (dark slate background, gold trend line, three bars, gold-
ringed accent dot). electron-builder picks it up automatically and
bakes it into the NSIS installer and the packaged .exe.

To regenerate the icon (e.g., after color or layout tweaks):

```
python scripts/generate-icon.py
```

Optional installer artwork that electron-builder will use if
present:

- `installerIcon.ico` / `uninstallerIcon.ico` — overrides the
  installer chrome icons; otherwise NSIS reuses `icon.ico`.
- `installerHeader.bmp` (150x57) and `installerSidebar.bmp`
  (164x314) — installer banner artwork.
